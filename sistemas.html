Mapeo de Sistemas para la Investigación Social desde el Diseño

Plataforma metodológica para comprender, visualizar y transformar sistemas complejos en proyectos de diseño social.

Fundamentos

¿Qué es el mapeo de sistemas?

El mapeo de sistemas es una metodología visual y analítica que permite representar cómo interactúan actores, procesos, recursos, barreras y resultados dentro de un fenómeno social complejo.

El objetivo central es **comprender la estructura profunda** de un sistema para identificar puntos de intervención, nodos críticos, dinámicas ocultas y oportunidades de transformación desde el diseño.

Conceptos esenciales

  • Sistema: conjunto de elementos interdependientes.
  • Actores: personas, instituciones, colectivos.
  • Flujos: movimiento de recursos, información o decisiones.
  • Nodos críticos: puntos donde el sistema se bloquea.
  • Palancas de cambio: lugares estratégicos donde intervenir.
Metodología

Metodología paso a paso

1. Delimitación del sistema

Define qué partes del fenómeno serán consideradas. Un sistema demasiado amplio pierde claridad; uno demasiado pequeño no revela relaciones significativas.

2. Identificación de elementos

Clasifica elementos en actores, recursos, procesos, barreras y resultados.

3. Representación de relaciones

Traza conexiones: quién influye a quién, qué flujo existe, qué depende de qué.

4. Detección de nodos críticos

Observa puntos donde el sistema se debilita: falta de recursos, roles desbalanceados, conflictos estructurales.

5. Identificación de palancas de intervención

Determina dónde una acción de diseño puede generar mayor impacto sistémico.

6. Visualización final

Construye un diagrama claro que pueda compartirse con comunidad, estudiantes o equipo interdisciplinario.

7. Revisión tras intervención

El sistema cambia con el tiempo; remapea para evaluar efectos reales.

Ejemplo aplicado

Relevo generacional en artesanía

1. Elementos identificados

  • Actores: jóvenes, maestros artesanos, autoridades, compradores.
  • Recursos: talleres, herramientas, ferias, redes.
  • Procesos: aprendizaje, producción, venta.
  • Barreras: ingresos bajos, migración, estigma social.

2. Flujos principales

Maestro → joven: transferencia de saber. Joven → feria: piezas para venta. Feria → ingreso: estabilidad económica.
